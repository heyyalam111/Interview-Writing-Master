# Interview Agent Team - 启动提示词

```
创建 3 人 Agent Team 帮我写采访稿。

Team Lead 职责：
- 先让风格研究员读取 style-reference/ 文件夹，分析写作风格
- 同时让故事挖掘师与用户进行访谈，挖掘故事细节
- 等两边都完成后，把风格指南+故事素材交给写手
- 写手完成初稿后，进行自检优化，最后输出 3 个标题选项 + 正文

队友职责：
1. 风格研究员：读取 style-reference/ 全部文章，分析语气、标题句式、开头写法、金句密度、段落节奏，输出风格指南
2. 故事挖掘师：与用户进行多轮访谈，根据回答动态追问，挖掘背景、冲突、细节、情感、升华，输出故事素材包
3. 写手：根据风格指南+故事素材，撰写 1000 字以内文章，确保每 300 字至少 1 句金句，输出 3 个标题选项+正文

注意：采访稿是商业人物访谈，需要吸睛、有故事张力。
```
